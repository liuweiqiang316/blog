# 命令式与声明式

命令式理论上要性能优于声明式，声明式代码可维护性更强

# 虚拟dom与innerHTML

innerHTML是销毁所有dom元素，再全量创建新的dom元素；

虚拟dom是先Diff比对，再部分替换。所以页面越复杂，相对来说虚拟dom的性能更好。

# 运行时与编译时

有template模板，则需要编译时；只有render函数就只需要运行时。
